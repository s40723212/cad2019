Webots 用戶指南

課程一 . 創建一個新的虛擬環境

本章節是讓你熟悉webot介面及操作，你將會創造一個虛擬空間包含了地板、牆壁， e-puck 機器人( e-puck robot )及 e-puck robot 控制系統。

如果你尚未安裝webot ，請參考這裡

執行webot

第一步驟:透過點擊webots圖示或使用終端命令來執行webots。

第二步驟:點擊來停止場景模擬，並依照路徑wizards/New Project Directory...創建新的虛擬環境，將儲存路徑及檔案名稱更改為my_first_simulation而不是建議名稱my_project。

第三步驟:展開RectangleArena將其中的floorTiliesizet及wallHight尺寸依序改為(0.25，0.25)、(0.05)。

第四步驟:點擊add選擇PROTO nodes (Webots Projects) / objects / factory / containers / WoodenBox (Solid)來新增木箱

1.將size尺寸更改成0.1 0.1 0.1

2.將translation尺寸更改為0 0.05 0

3.點擊shift將木箱拖曳至角落

4.使用contral+V複製木箱並按照3將木箱拖曳至你想要的位置

新增e-puck robot

第五步驟:依照路徑Wizards / New Robot Controller...新增控制器，並將其檔案命名為e-puck_go_forward1u

第六步驟:將原始碼貼上
